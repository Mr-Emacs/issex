# ISSEX

Issue tracker of doom. It works with a unique format.

## Build for linux

```bash
$ cc -o nob nob.c # or clang
$ ./nob
```

## Build for Windows

```bash
$ clang -o nob.exe nob.c # or clang
$ ./nob.exe
```

## To install

Run make
```bash
$ sudo make install
```
